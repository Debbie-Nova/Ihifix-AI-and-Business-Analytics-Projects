Retail Analytics Assignment

Overview

This repository contains my submission for the Retail Analytics Assignment. The objective of the assignment was to apply the RCTTCE (Role, Context, Task, Format, Constraints, Example) prompt engineering framework to generate a Stakeholder Brief for a retail company's CEO based on a business scenario involving declining performance across 30 stores.

Repository Contents

- RCTTCE Prompt – A structured AI prompt designed to guide the analysis and generation of a Stakeholder Brief.
- Stakeholder Brief – An AI-generated executive summary highlighting key business questions, decision areas, available data, and recommended next steps.
- Reflection – A brief explanation of the rationale behind the prompt design and structure.

Assignment Objective

To demonstrate the ability to:

- Design effective AI prompts using the RCTTCE framework.
- Generate business-focused insights from a retail analytics scenario.
- Communicate findings in a professional format suitable for executive stakeholders.
